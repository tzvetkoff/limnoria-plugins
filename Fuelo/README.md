# Fuelo

Fuelo: Fetches gasoline/diesel prices.

## Configuration

| Property                  | Description                                                    |
| ------------------------- | -------------------------------------------------------------- |
| **plugins.fuelo.apiKey**  | API key.  Network-specific, channel-specific.                  |
| **plugins.fuelo.timeout** | Fetch timeout in seconds.  Network-specific, channel-specific. |

## Usage

```
fuelo diesel
=> 1.66 €/л
```
